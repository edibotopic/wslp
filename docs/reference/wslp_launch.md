## wslp launch

Launch an interactive shell for a WSL distribution

### Synopsis

Launch an interactive shell for the specified WSL distribution.

This opens the default shell for the distro in the current terminal window.
The command will block until you exit the shell.

```
wslp launch <distro> [flags]
```

### Options

```
  -h, --help   help for launch
```

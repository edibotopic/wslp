## wslp serve

Start the HTTP API server

### Synopsis

Starts an HTTP server that exposes WSL operations via REST API for the Flutter GUI.

```
wslp serve [flags]
```

### Options

```
  -h, --help          help for serve
  -p, --port string   Port to run the server on (default "8080")
```

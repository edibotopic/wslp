# WSL Plus (`wslp`)

An experimental management tool for the Windows Subsystem for Linux
(WSL).

Provides simple-to-use actions for managing WSL distributions, using a
terminal or graphical interface.

Offers extended functionality compared to `wsl.exe`, including:

* Bulk actions (install, delete, backup)
* Single-step backup
* Single-step rename

Built using Go and Flutter.
